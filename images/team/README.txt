HOW TO ADD TEAM PHOTOS
======================

Drop JPEG files into this folder with these exact names:

  adeel-ahmad.jpg       → Prof. Dr. Adeel Ahmad
  iqra-komal.jpg        → Iqra Komal
  sadia-bano.jpg        → Syeda Sadia Bano Bukhari
  nimra-sarwar.jpg      → Nimra Sarwar
  muhammad-mehdi.jpg    → Muhammad Mehdi
  bilal-tipu.jpg        → Hafiz Muhammad Bilal Tipu
  hamna-habib.jpg       → Hamna Habib

Recommended size: 400×400 px, square crop, face centered.
If a photo is missing, the card shows the person's initials on a green background automatically.

To add a new team member:
1. Add their photo here as <id>.jpg
2. Open src/data/team.js and add a new entry following the same format.
